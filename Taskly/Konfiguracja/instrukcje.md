# Konfiguracja Taskly

1. Skopiuj plik `Taskly/Backend/.env.example` do `Taskly/Backend/.env` i uzupełnij.
2. Utwórz w Supabase tabele `profiles` i `tasks` zgodnie z README.
3. Włącz logowanie magic link oraz tradycyjne hasło według preferencji.
4. Skonfiguruj SMTP i wpisz dane w `.env`.
5. Uruchom `npm run dev` i odwiedź `http://localhost:3000/Html/logowanie.html`.
